About JavaScript

JavaScript was created by Brendan Eich in 1995.

It was developed in just 10 days for Netscape Navigator.

Originally, it was called Mocha, then LiveScript, before becoming JavaScript.

In 1996, JavaScript was submitted to ECMA for standardization.

Over time, it has become one of the most popular languages for web development.

console.log("Hello, World!")
Introduction to JavaScript

JavaScript is a programming language used for web development.

It runs in browsers to create interactive websites.

JavaScript can modify HTML and CSS elements dynamically.

It supports both client-side and server-side development.

With JavaScript, you can build responsive and user-friendly interfaces.

alert("Hello, World!");

Examples of function codes in JavaScript:

  • function add
  • function isEven
  • function greet

About to Java

Java was developed by James Gosling at Sun Microsystems in 1995.

It was initially called Oak, but later renamed Java.

Java was designed to be platform-independent, running on the Java Virtual Machine (JVM).

Its "write once, run anywhere" capability made it popular for various applications.

In 2010, Oracle acquired Sun Microsystems and took over Java's development.

CSS allows you to customize fonts, colors, spacing, and more.

Introduction to Java

JavaScript is a scripting language for web development.

Functions are reusable blocks of code.

System.out.println("Hello, World!")

They can take inputs, called parameters, and return values.

JavaScript vs Java

JavaScript is primarily used for web development, while Java is used for various applications.

JavaScript runs in browsers, whereas Java runs on the Java Virtual Machine (JVM).

JavaScript is interpreted, but Java is compiled into bytecode.

JavaScript is weakly typed, while Java is strongly typed.

JavaScript is event-driven and mainly client-side, while Java is class-based and supports both client and server-side.

console.log("Hello, World!"); vs. System.out.println("Hello, World!");
